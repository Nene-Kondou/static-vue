3-1
コンポーネントは再利用可能な「部品」

コンポーネントの作成はcomponentメソッドを呼びだす
　アプリケーション.component( 名前, { 設定情報 } )

②
Vueオブジェクトのように用意した値変数に収め、コンポーネントに渡して表示することもできる
アプリケーション.component( 名前, { 
    data() {
        return {
            message: ''
        }
    },
    template: '<>{{ message }}</>'
 } )

③属性の利用
「タグの属性」は、コンポーネントに「props」というプロパティとして用意しておく
コンポーネントのメッセージをコンポーネントのタグに用意
props: [名前1, 名前2, ...]
→使い方　<componentName 名前1="hello" 名前2="hello2" />
